# 🎲 我的 TurboWarp 自用扩展

---

## 简单说一下
这是我自己做的一个扩展，专门给 TurboWarp 用的，把平时做游戏、做动画常用的功能都整合在一起了，用着挺顺手，就发出来了。

---

## 里面都有啥？
### $\color{#FF9800}{🎲 各种随机功能}$
- 0到1的纯随机小数（无位数限制，做概率、倍率超好用）
- 自定义范围的随机数，比如-50到50、1.5到3.8这种
- 带概率的随机真假，默认50%，填10就是10%概率触发，做暴击、掉落很方便
- 可以存变量的随机角度！原版只能直接转向，这个能存起来反复用
- 自动适配舞台大小的随机X/Y，不管你改成多大的画布都能用
- 从文本里随机抽字符，能选抽几个，还能开/关重复抽取

### $\color{#2196F3}{👾 跨角色控制（这个是我觉得最实用的）}$
- 能直接读别的角色的X、Y、方向、显隐状态
- 能控制别的角色移动、转向、显示/隐藏
- 角色用下拉菜单选，默认就是你当前编辑的角色，不用打字，不会输错

---

## 怎么用？
1.  下载打开 TurboWarp
2.  左下角点「添加扩展」，拉到最下面选「加载扩展文件」
3.  选这个仓库里的 `random.js` 文件，加载完就能用了
