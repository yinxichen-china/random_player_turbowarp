# 🎲 TurboWarp 高级随机 & 跨角色控制扩展

$\color{#2196F3}{✨ 这是一个专为 TurboWarp 打造的超强原创扩展！}$

## 🎯 扩展功能一览

$\color{#FF5722}{📏 随机类功能}$
- 0~1 随机小数
- A~B 任意范围随机小数
- 自定义概率随机布尔
- 可储存随机角度
- 自适应舞台随机 X / Y 坐标
- 文本随机抽取字符（可重复开关）

$\color{#4CAF50}{👾 跨角色控制}$
- 读取其他角色的 X、Y、方向、显示状态
- 直接设置其他角色坐标、方向
- 显示 / 隐藏任意角色
- 角色下拉菜单，默认选中当前角色

$\color{#9C27B0}{💡 特色亮点}$
- ✅ 自适应任意舞台大小
- ✅ 原创实用，官方没有
- ✅ 下拉菜单选择角色，零错误
- ✅ 轻量流畅，不卡顿

---

## 🚀 使用方法
1. 打开 TurboWarp
2. 加载扩展 → 选择本 JS 文件
3. 直接使用所有超强功能！
